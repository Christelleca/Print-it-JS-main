# Print-It WebSite v.1

Print It - Bannière de Site Web:

Bienvenue dans le projet de bannière de site web pour Print It. Cette bannière de site web est conçue pour attirer l'attention des visiteurs et mettre en valeur les services exceptionnels de Print It.

Description:
Print It Banner est un composant interactif de carrousel d'images destiné à être intégré à votre site web grâce à JavaScript.

Fonctionnalités:
Carrousel interactif : Permet aux utilisateurs de faire défiler les différentes images du carrousel.
Flèches de navigation : Des flèches gauche et droite permettent de passer facilement d'une image à l'autre.
Points indicateurs : Des points sous la bannière indiquent le nombre d'images dans le carrousel et savoir ou nous sommes situé dans le carrousel.
Boucle infinie : Le carrousel continue de naviguer d'un slide à un autre, offrant une expérience utilisateur fluide.

Utilisation
Intégration dans votre site web : Incluez le code HTML, CSS et JavaScript.
Personnalisation des images et du texte : Ajoutez ou remplacez les images dans le tableau slides du fichier JavaScript ainsi
que la balise <p> pour adapter le texte à l'image du slide avec tagLine.
